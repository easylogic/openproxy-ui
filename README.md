# openproxy-ui 

Reverse Proxy for Front-End Developers 

# ElectronJS based 

# Install

```
git clone https://github.com/easylogic/openproxy-ui.git 
cd openproxy-ui 
npm install 
npm start 
  
```  

# Documents 

https://easylogic.github.io/openproxy-ui/
